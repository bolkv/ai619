# Multi-Organ Segmentation (`multi_organ_seg`)

**Model files**: `tools/multi_organ_seg/weights/nnunet/.../fold_2/multi_organ_seg.pth` + `tools/multi_organ_seg/weights/sam/sam_vit_h_4b8939.pth`

Simultaneously segments 16 organs from 3D abdominal CT using nnU-Net v2 + MaskSAM.

## Dependencies

`torch`, `torchvision`, `nibabel==5.4.2`, `numpy==1.26.4`, `omegaconf==2.3.0`, `scipy==1.13.1`, `scikit-image==0.23.2`, `scikit-learn==1.5.1`, `SimpleITK==2.5.3`, `pandas==2.3.3`, `tqdm==4.67.3`, `batchgenerators==0.25.1`, `dynamic-network-architectures==0.3.1`, `acvl-utils==0.2.5`, `einops`, `fvcore`, and more.

```bash
pip install -r requirements.txt
```

> `torch` and `torchvision` are not included in `requirements.txt` ‚Äî install them separately with the appropriate CUDA index.
>
> nnU-Net v2 is vendored under `vendor/nnunetv2/` and does not require a separate pip install. `infer.py` adds it to `sys.path` automatically at import time.

## Input

- **Format**: Single-channel 3D CT NIfTI file (`.nii` / `.nii.gz`)
- **Payload key**: `nifti_path`

## Output

- **Format**: 16-class NIfTI segmentation mask
- **Classes**:
  1. Spleen
  2. Right Kidney
  3. Left Kidney
  4. Gallbladder
  5. Esophagus
  6. Liver
  7. Stomach
  8. Aorta
  9. Inferior Vena Cava
  10. Pancreas
  11. Right Adrenal Gland
  12. Left Adrenal Gland
  13. Duodenum
  14. Bladder
  15. Prostate/Uterus
- **Location**: `results/<timestamp>/`

## Usage

```python
from tools.multi_organ_seg.logic import execute

result = execute({
    "nifti_path": "tools/multi_organ_seg/dataset/nnunet_raw/Dataset052_AMOS22_OnlyCT/imagesTs/amos_0600_0000.nii.gz",
})

print(result["summary"])
print(result["artifacts"]["mask_path"])
```

### Payload Fields

| Key | Type | Description |
|---|---|---|
| `nifti_path` | `str` | Path to a single-channel 3D CT NIfTI file |
| `device` | `str` | `"gpu"` (default) or `"cpu"` |
| `output_dir` | `str` | Override output directory |
| `overrides` | `list[str]` | OmegaConf key=value overrides |
| `file_name` | `str` | Display name for results |

## Configuration

Default settings in `config.yaml`:

| Key | Value |
|---|---|
| `dataset_name` | `Dataset052_AMOS22_OnlyCT` |
| `trainer` | `MaskSAM_AMOS` |
| `plans` | `nnUNetPlans` |
| `configuration` | `3d_fullres` |
| `fold` | `2` |
| `checkpoint_name` | `checkpoint_final.pth` |
| `tile_step_size` | `0.5` |
| `use_mirroring` | `false` |
| `perform_everything_on_device` | `true` |

## Limitations

- Very high VRAM usage (SAM ViT-H + nnU-Net loaded simultaneously)
- TTA mirroring is disabled (`use_mirroring: false`) to save memory
- Weights cannot be auto-downloaded ‚Äî manual placement required
- Only validated on AMOS 2022 CT data
